# 📖 MyDiary

一款简洁优雅的日记应用，支持心情记录、图片插入、标签分类、密码锁等功能。

## ✨ 功能特性

### 📝 日记管理
- 创建、编辑、删除日记
- 支持标题 + 正文 + 标签
- 置顶日记
- 批量操作（多选删除、移动文件夹）
- 最后编辑时间记录
- 未保存修改提醒

### 🎭 心情日记
- 10 种心情 emoji（😊😌🥰🤔😴😢😤🥳😰😶）
- 日历视图心情颜色标记
- 导出卡片显示心情

### 📷 图片插入
- 每篇日记最多 9 张图片
- 支持从相册选择、删除图片
- 图片保存到相册

### 📁 文件夹管理
- 自定义文件夹（颜色、图标）
- 文件夹筛选
- 🔒 文件夹密码锁（安全存储）

### 📅 日历视图
- 月视图日历
- 有日记的日期标记
- 点击日期查看当天日记
- 年月选择器

### 📊 数据统计
- 总篇数、本月篇数
- 总字数统计
- 连续写日记天数

### 🎨 个性化
- 🌙 深色模式（浅色/深色/自动）
- 字号设置（小/中/大/特大）
- 字体设置（系统/宋体/黑体/楷体）
- 头像和昵称

### 📤 导出 & 导入
- **导出格式**：JSON / HTML / Markdown / TXT
- **选择性导出**：两级选择（文件夹 → 日记）
- **Markdown 导出**：带 YAML frontmatter，支持重新导入
- **日记卡片**：生成精美图片分享

### 🗑️ 回收站
- 软删除，30 天内可恢复
- 永久删除 / 清空回收站

## 🛠️ 技术栈

- **框架**：React Native + Expo SDK 54
- **导航**：React Navigation 7（Stack + Bottom Tabs）
- **数据库**：expo-sqlite
- **安全存储**：expo-secure-store
- **图片**：expo-image-picker / expo-media-library
- **分享**：expo-sharing / react-native-view-shot

## 🚀 快速开始

```bash
# 克隆项目
git clone https://github.com/wzhwill331/MyDiary.git
cd MyDiary

# 安装依赖
npm install

# 启动开发服务器
npx expo start

# 用 Expo Go 扫码运行
```

## 📱 运行环境

- Expo Go（推荐）
- React Native 0.81+
- iOS 15+ / Android 6+
